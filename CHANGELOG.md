# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-03-12

### Added

- Expo Router file-based navigation
- NativeWind styling with dark theme design system
- Supabase Auth integration with SecureStore
- Zustand stores (auth, onboarding)
- TanStack Query + Axios API layer
- 7 design system components (Button, TextInput, Card, ProgressBar, StepIndicator, ChipSelector, ToggleOption)
- Auth screens (login, signup, forgot password)
- Role selection screen
- 8-step onboarding form
- Client tab group (Home, Training, Chat, Meals, Profile)
- ESLint 10 flat config + Prettier
- Husky + commitlint for conventional commits
